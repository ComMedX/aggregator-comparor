Welcome to Aggregator Comparor

This program lets you browse the Aggregator Advisor database and cross reference an additional ligand database for similar structures. To load a ligand database please use the manage.py script on the command line.

Aggregators

Browse and search for aggregators either by name or chemical structure.

Ligands

Browse and search for ligands (from your database of choice) either by name or chemical structure.

Citations

Browse citations in the aggregator advisor and view aggregators by original identifying paper.
